# What is the Enjin Web Wallet?

The Enjin Web Wallet offers a simple and accessible solution for users who want to dip their toes into the world of NFTs. It serves as a non-custodial onboarding tool, allowing users to learn about NFTs at their own pace before moving to a full-featured wallet application. As a temporary storage solution, the Web Wallet offers a fast and user-friendly web-based platform for claiming and collecting NFTs. It supports various file formats, including JPG, PNG, GIF, and MP4, for both ERC-1155 Ethereum Mainnet and Enjin NFTs, making it a versatile tool for users who don't yet have a blockchain wallet.

# How Does the Enjin Web Wallet Work?

The Enjin Web Wallet allows users to claim tokens instantly using Enjin Beam and use the browser as a temporary wallet. The wallet serves as a convenient entry point to non-custodial solutions, allowing users to migrate into the Enjin Wallet as one of their addresses using a 12-word recovery phrase. Tokens stored in the web wallet can be sent anywhere the user wants, making it a versatile tool for managing token storage. The process of migrating the web wallet to the Enjin Wallet is similar, making it easy to manage your assets across both platforms.

# The Web Wallet Features:

- **Easy NFT access: **With Web Wallet, you can easily access all your NFTs on the web.
- **Powered by Enjin Beam: **Built with Enjin Beam technology for distribution and turbocharged with Enjin's scalability.
- **User-friendly: **Web Wallet offers a hassle-free way to collect NFTs, even for beginners.
- **Completely web-based: **No need to download any app or generate a wallet ID to start collecting your favorite NFTs.
- **Branded NFT distribution: **Fully customizable for enterprise or specific case-by-case projects.
